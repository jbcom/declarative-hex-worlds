    Module src/manifest/upstream-layout

    Typed description of the KayKit Medieval Hexagon Pack upstream layouts.

    Remarks

    The library bootstraps assets at install time by mirroring the upstream GitHub source tree (or a user-supplied zip with identical layout) into the consumer's asset root. Both editions share the same shape: a top-level pack directory containing Assets/, Textures/, Samples/, marker files (License.txt, Medieval_Hexagon_UserGuide_v1.pdf, contents_*.jpg), and social .url shortcuts. Asset binaries live under Assets/<format>/... where format is one of gltf, fbx, fbx(unity), or obj.

    Only the gltf tree is mirrored by the bootstrap step. Source-format dirs (fbx, obj) are filtered out unless includeSourceFormats is set.
